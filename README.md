# Fashion MNIST
CNN Classification of the famous fashion MNIST dataset
## Dataset
The dataset is taken from the [tensordash.keras.datasets](https://www.tensorflow.org/api_docs/python/tf/keras/datasets/fashion_mnist/load_data) module.
